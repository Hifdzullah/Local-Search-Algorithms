# 8-queens-NxN-chess-puzzle-using-6-algorithms-

 Solving 8-queens on N x N chess puzzle using six different algorithms
 
1. Stohastic Hill Climbing
2. Hill Climb
3. Hill Climb Random Restart
4. Local Beam Search 
5. Genetic Algorithm
6. Random Walk
